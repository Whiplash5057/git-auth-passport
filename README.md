# o-AUTH - SIMPLE LOGIN AND SIGNUP For Instagram and Google+


### Steps to run code:
1. npm run dev   ( To start the server )
2. open http://localhost:3090

### Plugin dependencies:

1. body-parser
2. cookie-session
3. ejs
4. express
5. mongoose
6. morgan
7. passport
8. passport-google-oauth20
9. passport-instagram
